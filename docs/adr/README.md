# Architecture Decision Records

Records of consequential, hard-to-reverse decisions made in this folder, and the
reasoning behind them — so future work doesn't relitigate or accidentally undo them.

ADRs are append-only. To change a past decision, add a new ADR that supersedes it;
do not edit the old one. New decisions get the next sequential number.
