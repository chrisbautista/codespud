Hi, I'm Chris Bautista — a frontend engineer who specializes in accessible, production-grade web interfaces. <!--more-->

Most of my recent work sits at the intersection of accessibility (WCAG 2.1 AA) and modern frontend engineering: auditing components for screen reader and keyboard support, fixing real-world issues like focus traps, missing alt text, and unlabeled controls, and writing about what I learn along the way so other developers can skip the trial and error.

## What I work with

  * **Languages:** JavaScript, TypeScript
  * **Frontend:** React, Gatsby, Tailwind CSS
  * **Accessibility:** WCAG 2.1 AA audits, ARIA patterns, keyboard/screen-reader testing, semantic HTML
  * **Testing:** Vitest
  * **Other:** Node.js, REST APIs, CSS (container queries, modern layout)

## Recent case studies

  * [Build an Accessible Tab Container](/accessible-tab-container/) — roving tabindex, arrow-key navigation, ARIA tab pattern
  * [Fix accessibility using focus-within](/fix_accessibility_with_focus_within/) — solving keyboard-vs-mouse UI mismatches without JavaScript
  * [Automatically generate skip buttons](/skip-button-js/) — scripted skip-navigation links
  * [Accessible UI Component Series](/accessible-ui-component-series/) — an accordion, buttons, and shared design tokens built to test ARIA patterns firsthand

Also see [Creating My Animated Bottom Bar Experiment](/animated-bottom-bar/), my most-loved CodePen experiment — interaction and motion rather than accessibility, but a good look at range.

See the full [portfolio](/works) or browse posts tagged [accessibility](/tags/accessibility) for more.

## A bit of background

I've been building for the web for over 15 years — from hybrid mobile apps (Cordova, jQuery Mobile) and PHP/REST backends to, more recently, React and TypeScript. These days accessibility is the thread that ties most of my work together.

I still own [CodeSpud Web Services][1], and you can find my code on [GitHub][3]. Have a challenge I can help with? Post it as a comment or email me at <u>**challenges at codespud dot com**</u>.

Outside of code: gaming, cooking, and chasing the perfect cup of coffee.
